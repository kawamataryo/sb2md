# Scrapbox2Markdown

Scrapbox記法をMarkdown記法に変換するBookmarklet．

### Minify

```
$ brew install closure-compiler
$ closure-compiler --js main.js --js_output_file main.min.js
$ closure-compiler --js tex.js --js_output_file tex.min.js
```

### esa用
